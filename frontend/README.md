# Atlas Paises Pj2

Atlas Paises Pj2 e uma aplicacao web SPA desenvolvida com React para exploracao de dados de paises em tempo real.

O projeto consome um backend Express com PostgreSQL e apresenta informacoes como nome, capital, regiao, populacao, area e bandeira.

## Tecnologias

- React
- useReducer para gerenciamento de estado
- Material UI para interface
- Axios para requisicoes HTTP
- Vite para ambiente de desenvolvimento e build

## Funcionalidades

- Busca de paises por nome
- Filtro por regiao
- Ordenacao por nome, populacao e extensao territorial
- Modal com detalhes completos do pais
- Favoritos com persistencia em localStorage

## Objetivo Academico

Este projeto atende ao escopo da disciplina de Programacao Web Fullstack, aplicando os conceitos de Single Page Application, consumo de API propria e desenvolvimento frontend moderno com React.

## Execucao

Escolha uma das opcoes abaixo para iniciar o projeto.

### Opcao 1: Start manual

1. Instale as dependencias:

	npm install

2. Inicie o servidor de desenvolvimento:

	npm run dev

3. Abra no navegador:

	http://localhost:5173

### Opcao 2: Start automatico (Windows)

1. Execute o arquivo abaixo na raiz do projeto:

	iniciar-app.bat

2. O script instala as dependencias automaticamente (se necessario) e inicia o servidor.

3. Abra no navegador:

	http://localhost:5173
