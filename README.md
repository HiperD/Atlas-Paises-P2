# Atlas Paises Pj2

Projeto fullstack para a disciplina de Programacao Web Fullstack, com frontend em React/Vite e backend em Express + PostgreSQL.

## Estrutura

- `frontend/`: aplicacao React consumindo a API propria
- `backend/`: API Express com login, busca e insercao

## Como rodar

1. Inicie o backend dentro de `backend/` com `npm run dev`.
2. Inicie o frontend dentro de `frontend/` com `npm run dev`.
3. Use o login `admin` e senha `123456` para acessar a aplicacao.
